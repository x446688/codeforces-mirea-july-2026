# Входные данные
Первая строка содержит одно целое число 𝑡 — количество наборов входных данных (1≤𝑡≤10000). Далее следуют описания самих наборов.

Первая строка каждого набора входных данных содержит одно целое число 𝑞 (1≤𝑞≤200000) — количество запросов.

Следующие 𝑞 строк содержат запросы в одном из следующих форматов:

- `push 𝑥`, где 𝑥 — целое число (−109≤𝑥≤109);
- `pop`;
- `get 𝑖`, где 𝑖 — целое число (1≤𝑖≤1018).

Гарантируется, что перед выполнением каждого запроса `pop` и `get` массив не пуст.

Гарантируется, что сумма 𝑞 по всем наборам входных данных не превосходит 200000.

# Выходные данные
Для каждого запроса выведите одно целое число в отдельной строке. Для запроса `push 𝑥` выведите новую длину массива, для запроса `pop` — удалённый элемент, для запроса `get 𝑖` — значение элемента на позиции 𝑖 в бесконечном повторении текущего массива.
