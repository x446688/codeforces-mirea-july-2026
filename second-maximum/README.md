# Входные данные
Первая строка содержит одно целое число 𝑡 — количество различных наборов входных данных (1≤𝑡≤10000). Далее следуют описания самих наборов.  


Первая строка каждого набора входных данных содержит одно целое число 𝑛 (2≤𝑛≤200000) — длину массива.  

Вторая строка каждого набора входных данных содержит 𝑛 целых чисел a_1, a_2,..., a_n (−109≤a_i≤109).  
Гарантируется, что сумма 𝑛 по всем наборам входных данных не превосходит 200000.
# Выходные данные
Для каждого набора входных данных выведите одно целое число — второй максимум в этом массиве.
